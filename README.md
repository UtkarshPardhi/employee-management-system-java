# Employee Management System

A console-based Java application using JDBC and PostgreSQL
to perform CRUD operations on employee records.

## Features
- Add Employee
- View Employees
- Update Employee
- Delete Employee
- Search by Department

## Technologies
- Java
- JDBC
- PostgreSQL
- Git & GitHub

## Project Structure
src/
├── DBUtil.java
├── Employee.java
├── EmployeeController.java
├── EmployeeRunner.java
└── LoggerUtil.java

## How to Run
1. Create database and table in PostgreSQL
2. Update DB credentials in DBUtil.java
3. Run EmployeeRunner.java
